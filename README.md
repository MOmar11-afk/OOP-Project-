# Employee Management System

A console-based Employee Management System developed in C++ using
Object-Oriented Programming principles and File Handling.

## Features
- Add Employee
- Update Employee
- Delete Employee
- Search Employee
- Persistent File Storage

## Technologies
- C++
- OOP
- File Handling
- VS Code

## How to Run

1. Clone the repository
2. Open in Visual Studio Code
3. Compile using g++
4. Run the executable
